## 세계관 / 스토리

- **배경**: 2024년 대한민국, 저출산으로 마법소녀 부족 현상 발생
- **마법청**: 정부가 신설한 기관. 성별/연령 제한을 없애고 마력을 가진 모든 인물을 모집
- **주인공**: 김부장 — 렐루보험 영업부의 평범한 중년 남성 직장인. 어릴 때부터 마법소녀를 꿈꿨으나 포기했다가, *냥을 만나 마법소녀가 됨
- **핵심 갈등**: 창피함 vs 세계를 지켜야 하는 사명감 (민망한 주문을 외쳐야만 마법 발동)

## 핵심 게임플레이

### 기본 흐름

```
스토리 진행 → 적 등장 → 변신 주문 시전 → 배틀 시작 → 주문 읽기(공격) → 방어 주문 → 반복 → 승리/패배
```

### 주문 시스템

- 화면에 마법 주문 텍스트가 표시됨
- 플레이어가 **마이크에 대고 직접 소리 내어 읽어야 함**
- 주문 예시:
  - 변신 주문: *"니코니코 웃음, 마음을 환하게! 뽀옹뽀옹 하트 스마일, 뽀빠뽀빠 하트웨이브!"*
  - 공격 주문: *"샤랄라 예뻐질래 꿈꿔왔던 내 모습, 마법소녀로 변신!"*
  - 무의미 조합: *"파이넌셜 파이어스톰 테러"* 같은 병맛 주문도 존재
- **변신 주문**은 무지개색으로 표시, 일부 글자는 회전하는 연출
- 주문은 감정을 배제하고 **또박또박** 말해야 하는 것부터, **감정을 실어서 높낮이를 확실히** 해야 하는 것까지 다양

### 게이지 시스템

- 주문을 읽으면 **게이지 바**가 채워짐
- **50% 이상 채워야** 마법 발동 — 미달 시 마법 실패, 턴 넘어감
- 게이지를 **많이 채울수록 공격력 상승**
- 실패한 주문은 성공할 때까지 재시도 필요

### AI 음성 평가 기준

렐루게임즈 자체 개발 AI가 다음을 **종합 평가**하여 대미지 계산:

1. **목소리 크기** — 단, 단순히 소리만 지르면 안 됨
2. **발음 정확도** — 주문과 다른 글을 읽으면 감지됨
3. **감정 표현** — 높낮이를 확실히 주면 대미지 증가

> 핵심: 소리를 크게만 한다고 위력이 올라가지 않음. **감정을 실어서 정확하게** 읽어야 함.

### 방어 시스템

- 상대의 강공격 시 **5초 안에 방어 주문**을 외워야 함
- 상황에 따라:
  - 빠르게 주문을 외워 **쉴드 전개**
  - 아주 큰 소리로 외워 **위력 증폭**

### 승리 조건

- 상대방의 **정신력(HP)을 0**으로 만들면 승리

## 대전 모드

### 스토리 모드 (PvE)

- 에피소드별 스토리 진행
- NPC 적과 음성 배틀
- 각 에피소드마다 고유한 캐릭터와 특수 기믹

### PvP 온라인 대전

| 항목 | 내용 |
|---|---|
| 매칭 방식 | **7자리 코드 공유** 또는 **랜덤 매치** (2024.09 추가) |
| 대전 방식 | 양측 **동시에** 주문 시전 |
| 단어 선택 | 각자 주어진 **6개 단어 중 1개 선택** |
| 주문 생성 | 선택된 2개 단어를 **조합하여 마법 주문 생성** |
| 주문 시전 | 두 사람이 같은 주문을 외침 |
| 제한 사항 | 같은 언어 사용자끼리만 매칭 |

### 로컬 대전

- **핫시트 방식**: 양측이 **돌아가면서** 주문을 외침 (한 기기, 한 마이크)

## 연출 / UI 특징

- 주문 시전 시 게임이 자동으로 **에코 효과** 추가
- 플레이어의 음성을 **재생**해줌 (자기 목소리 듣기)
- 변신 주문: **무지개색 텍스트 + 글자 회전** 효과
- 마법소녀 변신 씬 연출
- 병맛 유머와 진지한 배틀이 공존하는 톤

## voice-battle과의 유사점 / 오마주 포인트

| 루루핑 요소 | voice-battle 대응 | 오마주 가능성 |
|---|---|---|
| 마이크로 주문 시전 | 마이크로 명대사 녹음 | 핵심 메커니즘 동일 |
| AI 음성 평가 (크기/발음/감정) | Gemini AI 심판 (점수 0~100) | 유사 — AI가 음성 판정 |
| 정신력(HP) 시스템 | HP 100 시스템 | 거의 동일 |
| 게이지 채우기 → 마법 발동 | 녹음 → AI 점수 → 데미지 | 유사 구조 |
| 7자리 코드 온라인 대전 | 6자리 코드 온라인 대전 | 거의 동일 |
| 6개 단어 중 선택 → 주문 생성 | 키워드 선택 → AI 문장 생성 | 유사 — 단어 기반 문장 생성 |
| 로컬 핫시트 + 온라인 PvP | 로컬 모드 + 온라인 모드 | 동일 구조 |
| 에코 효과 + 음성 재생 | 녹음 재생 기능 | 연출 오마주 가능 |
| 변신 주문 연출 (무지개, 회전) | 배틀 애니메이션 (7단계) | 연출 오마주 가능 |
| 병맛 주문 + 창피함 요소 | 한국 명대사 (감정 과잉 연기) | 톤 유사 |
